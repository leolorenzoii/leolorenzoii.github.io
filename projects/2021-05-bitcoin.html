The Bitcoin Network

May 2021

This project was done alongside my Term 4 MSDS 2021 SLT, Perry Ervine Ang.

Introduction

Cryptocurrency is one of the fastest-growing asset classes, growing by over 130% CAGR in the past eight years to achieve a total market capitalization of USD 1.5 trillion as of May 31, 2021, which is already six times larger than the USD 249 billion market capitalization of the Philippine Stock Exchange. This has led to a strong surge in the number of crypto investors from 5.8 million in the second quarter of 2017 to 101 million as of the third quarter of 2020.

Nevertheless, investing in cryptocurrencies can be challenging given that (1) it is not very straightforward to understand how it works, and (2) there are over 4000 cryptocurrencies available to invest in. These two factors make it difficult for new investors to choose which cryptocurrency to invest in. This project aims to address this by finding a way to systematically identify key cryptocurrencies without having to study how each one of them works.

Methodology

The objective of the project is to use the use-case descriptions of each cryptocurrency found in CoinMarketCap.com (CMC) as basis for the selection of a diversified cryptocurrency portfolio. The methodology of this project is outlined as follows:

We first scrape the cryptocurrency descriptions from CMC which we subsequently clean. In order to extract the meaning of each description, we use Latent Semantic Analysis to get the topics of each cryptocurrency. The cryptocurrency network is then constructed with the coins as the nodes of the network while the similarity of their descriptions as edges. Since each cryptocurrency is grouped by CMC using a tag, we obtain the subgraph of each tag then select the coin that best describes each tag using the degree centrality measure. Finally, we strategize which coin to prioritize by using different network metrics.

Results

With our methodology, we shortlisted 132 unique cryptocurrencies from the original set of 4620. This is still a sizeable set of names and as such, we identified four strategies to rank them to arrive at cryptocurrencies we should be prioritizing to study over others.

  1. Market capitalization of each coin central_coins: Based from the 132 selected coins, rank the coins according to their market capitalization.
  2. Centrality/Order co_coins: Normalize the centrality of the central coin of each tag subgraph with respect to the order (number of nodes) of the subgraph, then rank the coins based on this normalized centrality.
  3. Centrality/Size cs_coins: Normalize the centrality of the central coin of each tag subgraph with respect to the size (weighted sum of the edges) of the subgraph, then rank the coins based on this normalized centrality.
  4. Clustering Coefficient cc_coins: Rank the coins based on the average clustering coefficient of the subgraph that it belongs to.

To select which strategy would be the best, we compare the returns of these cryptocurrencies in the past 30 days, 60 days, and 90 days for the identified top 20 cryptocurrencies for each strategy. For reference, we also get the returns of cryptocurrencies with the highest market capitalization as cmc_coins, since picking the cryptocurrencies with the largest market capitalization is a logical baseline strategy for new investors.

Plotting the returns in a bar graph shows that the cryptocurrencies selected according to the size-normalized centrality measure cs_coins yield the best returns for most time periods.

Conclusion

In this project, we have shown how we can model cryptocurrencies as a network by using the similarity between their use-case descriptions as edges.

This allows us to identify key cryptocurrencies by looking at the central nodes in the network, which can be instrumental especially in finding lesser-known cryptocurrencies “gems” that may potentially deliver outsized returns.

We find that buying the top 20 cryptocurrencies identified with this framework will have generated a 73% to 446% return in the past 30 to 90 days, which outperforms investing in cryptocurrencies with the highest market capitalization by 53% to 321% over the same period.
